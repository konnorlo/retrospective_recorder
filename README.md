Retrospective Recorder VST3

Download:
1. Open this repo on GitHub.
2. Click Code -> Download ZIP.
3. Unzip it.
4. Use the files in:
   - mac/Retrospective Recorder_macOS.vst3
   - windows/Retrospective Recorder_windows.vst3

Files in this repo:
- mac/Retrospective Recorder_macOS.vst3
- windows/Retrospective Recorder_windows.vst3 (if included)

Install (macOS):
1. Copy the .vst3 bundle to:
   ~/Library/Audio/Plug-Ins/VST3/
2. Open your DAW and rescan plugins.

Install (Windows):
1. Copy the .vst3 file/folder to your VST3 plugin path.
   Default:
   C:\Program Files\Common Files\VST3\
2. Open your DAW and rescan plugins.
